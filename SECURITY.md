# Security Policy

## Supported Versions

Only the latest release on `main` is actively maintained.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report privately by emailing the maintainer (not provided, use 2nd method) (see the GitHub profile for contact), or use [GitHub's private vulnerability reporting](https://github.com/Lorakszak/wavern/security/advisories/new) if enabled on this repository.

Include:
- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any suggested mitigations

You will receive a response within 14 days.
